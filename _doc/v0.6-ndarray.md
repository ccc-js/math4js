# v0.6 - ndarray 實作

## 目標

實現一個 NumPy 風格的 ndarray 類別，移除對外部 ndarray 依賴，專為 math4js 的數值運算需求設計。

## 為何自訂？

1. **減少依賴**: 不需要 `ndarray` 或 `@stdlib/ndarray`
2. **更簡單**: 只實現需要的核心功能
3. **可控制**: 易於整合到 math4js
4. **NUMPY 相容**: API 與 NumPy 相似

## NumPy ndarray API

### 建構子

```typescript
ndarray.zeros([3, 4])           // 3x4 零矩陣
ndarray.ones([2, 3, 4])        // 2x3x4 全 1
ndarray.array([[1, 2], [3, 4]])  // 從 2D array
ndarray.fromFlat(data, [3, 4]) // 從扁平資料
ndarray.full([2, 3], 5)        // 全 5 矩陣
```

### 屬性

| 屬性 | 說明 |
|------|------|
| `.shape` | 維度 tuple |
| `.ndim` | 維度數 |
| `.size` | 總元素數 |
| `.dtype` | 資料型別 |
| `.T` | 轉置視圖 |
| `.flat` | 扁平迭代器 |

### 索引切片

```typescript
a[0]           // 第一列
a[0, 1]        // a[0][1]
a[0:2, 1:3]    // 子矩陣 (slice)
a[:, 0]        // 第一欄
a[::2, ::2]    // 間隔取值
```

### 產生器

```typescript
ndarray.arange(12)           // [0..11]
ndarray.arange(0, 10, 2)     // [0, 2, 4, 6, 8]
ndarray.linspace(0, 1, 5)    // [0, 0.25, 0.5, 0.75, 1]
ndarray.eye(3)               // 3x3 單位
ndarray.diag([1, 2, 3])      // 對角
ndarray.random.rand([2, 3])  // 均勻亂數
ndarray.random.randn([2, 3]) // 常態亂數
```

### 運算

```typescript
// 元素wise
a + b, a - b, a * b, a / b, a ** 2

// 矩陣乘法
a @ b

// 純量
a + 1, a * 2

// 比較
a > b, a == 5
```

### 聚合

```typescript
a.sum(), a.mean(), a.std(), a.var()
a.min(), a.max(), a.prod()
a.sum(0), a.sum(1)   // axis
```

### 形狀

```typescript
a.flatten()       // 攤平
a.reshape([2, 6])
a.T               // 轉置
a.swapaxes(0, 1)
a.squeeze()
a.expand_dims(0)
```

## 與 Matrix 整合

```typescript
// ndarray -> Matrix
const m = new Matrix(arr)

// Matrix -> ndarray
const arr = m.toNdarray()
```
