# v0.4 - 線性代數 (Linear Algebra)

## 目標

實現基於 numeric.js 的矩陣運算與矩陣分解功能，API 與 math4py/linear_algebra 保持一致。

## 依賴

- **numeric**: 矩陣底層計算（svd, lu, eig 等）

## 模組結構

```
src/math4js/linear_algebra/
├── index.js          # 匯出
├── matrix.js         # Matrix 類別
├── function.js       # 函數式 API (det, transpose, ...)
└── vector.js         # 向量運算
```

## API 設計（與 math4py 一致）

### Matrix 類別

```javascript
const { Matrix } = require('math4js/linear_algebra');

// 建立矩陣
const m = new Matrix([
  [1, 2],
  [3, 4],
]);
const I = Matrix.eye(3); // 單位矩陣
const Z = Matrix.zeros([2, 3]); // 零矩陣
const O = Matrix.ones([2, 3]); // 全 1 矩陣
const R = Matrix.random([3, 3], -1, 1); // 亂數

// 屬性
m.shape; // [2, 2]
m.T; // 轉置 (回傳新 Matrix)

// 運算
m.add(other); // 加法
m.sub(other); // 減法
m.mul(other); // 元素乘法 或 純量乘法
m.dot(other); // 矩陣乘法 @
m.inv(); // 反矩陣
m.det(); // 行列式
m.eig(); // { values, vectors }
m.eigvals(); // 特徵值陣列
m.svd(); // { U, S, V }
m.qr(); // { Q, R }
m.lu(); // { P, L, U }
m.solve(b); // 解 Ax = b
m.rank(); // 矩陣秩
m.norm(ord); // 範數
```

### 函數式 API

```javascript
const la = require('math4js/linear_algebra');

la.det(m); // 行列式
la.transpose(m); // 轉置
la.trace(m); // 跡
la.matrix_add(a, b); // 矩陣加法
la.matrix_multiply(a, b); // 矩陣乘法
la.matrix_scalar_mul(m, s); // 純量乘法
la.inverse_2x2(m); // 2x2 反矩陣
la.dot_product(u, v); // 內積
la.cross_product(u, v); // 外積
la.norm_vector(v); // 向量範數
```

## 實作原則

1. 使用 numeric.js 作為底層計算引擎
2. Matrix 類別包裝 2D Array，内部使用 numeric 格式
3. 支援鏈式呼叫（方法返回新 Matrix）
4. 錯誤處理與型別檢查
