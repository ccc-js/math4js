## 功能項目

### 1. 基礎函數 (function.ts) ✅

- [x] `gcd(a, b)` - 最大公因數 (歐幾里得算法)
- [x] `lcm(a, b)` - 最小公倍數
- [x] `isPrime(n)` - 質數判斷
- [x] `primesUpTo(n)` - 埃拉托斯特尼篩法
- [x] `primeFactors(n)` - 質因數分解
- [x] `eulerPhi(n)` - 歐拉 φ 函數
- [x] `modPow(a, b, m)` - 模冪運算
- [x] `modInv(a, m)` - 模反元素
- [x] `fibonacci(n)` - 費波那契數
- [x] `factorial(n)` - 階層
- [x] `combinations(n, k)` - 二項係數
- [x] `isCoprime(a, b)` - 互質判定

### 2. 定理驗證 (theorem.ts) ✅

- [x] `bezoutIdentity(a, b)` - 貝祖等式
- [x] `fundamentalTheoremOfArithmetic(n)` - 算術基本定理
- [x] `eulerPhiMultiplicative(m, n)` - φ 函數積性
- [x] `fermatLittleTheorem(p, a)` - 費馬小定理
- [x] `eulerTheorem(n, a)` - 歐拉定理
- [x] `chineseRemainderTheorem(m, a)` - 中國剩餘定理
- [x] `gcdLcmRelation(a, b)` - gcd*lcm = |ab|
- [x] `fibonacciGcdProperty(m, n)` - gcd(F(m),F(n)) = F(gcd(m,n))

## API 設計

```typescript
import { gcd, lcm, isPrime, primeFactors, eulerPhi, fibonacci, modPow, modInv } from 'math4js/number_theory';

// 最大公因數
gcd(48, 18); // 6

// 最小公倍數
lcm(4, 6); // 12

// 質數判斷
isPrime(17); // true

// 質因數分解
primeFactors(12); // [2, 2, 3]

// 歐拉 φ 函數
eulerPhi(12); // 4

// 費波那契
fibonacci(10); // 55

// 模冪運算
modPow(2, 10, 1000); // 24

// 模反元素
modInv(3, 11); // 4

// 貝祖等式
bezoutIdentity(48, 18); // { x: -1, y: 3, d: 6 }

// 中國剩餘定理
chineseRemainderTheorem([3, 4, 5], [2, 3, 1]); // 11
```
