## 功能項目

### 1. 單變數微分 (derivative.ts) ✅

- [x] `derivative(f, h?)` - 數值導數（中央差分）
- [x] `secondDerivative(f, h?)` - 二階導數
- [x] `partial(f, i, h?)` - 偏導數
- [x] `gradient(f, x, h?)` - 梯度向量
- [x] `jacobian(f, x, h?)` - Jacobian 矩陣

### 2. 單變數積分 (integral.ts) ✅

- [x] `trapezoid(f, a, b, n)` - 梯形法
- [x] `simpson(f, a, b, n)` - Simpson 法
- [x] `romberg(f, a, b, tol)` - Romberg 積分
- [x] `adaptive(f, a, b, tol)` - 自適應積分
- [x] `monteCarlo(f, a, b, n)` - Monte Carlo 積分
- [x] `gaussLegendre(f, a, b, n)` - Gauss-Legendre

### 3. 多變數微積分 (multivariable.ts) ✅

- [x] `gradient(f, x)` - 梯度 ▽f
- [x] `divergence(F, x)` - 散度 ▽·F
- [x] `curl(F, x)` - 旋度 ▽×F
- [x] `laplacian(f, x)` - 拉普拉斯算子 ▽²f
- [x] `hessian(f, x, h?)` - Hessian 矩陣
- [x] `directionalDerivative(f, x, direction)` - 方向導數

### 4. 數列與級數 (sequence.ts) ✅

- [x] `sequence(f, n0, n)` - 數列 f(n)
- [x] `series(f, a, b)` - 級數求和
- [x] `converge(f, x0, tol, maxIter)` - 收斂判定
- [x] `limit(f, a, dir?)` - 極限

### 5. 泰勒展開 (taylor.ts) ✅

- [x] `taylor(f, a, n)` - 在 a 點的 n 階展開
- [x] `maclaurin(f, n)` - 在 0 點展開
- [x] `seriesSum(f, a, n)` - 級數求和

### 6. 數值優化 (optimize.ts) ✅

- [x] `goldenSection(f, a, b, tol)` - 黃金分割法
- [x] `gradientDescent(f, df, x0, lr, tol)` - 梯度下降
- [x] `newtonMethod(f, df, ddf, x0, tol)` - 牛頓法
- [x] `conjugateGradient(A, b, x0, tol)` - 共軛梯度法

## API 設計

```typescript
import {
  derivative, integral,
  gradient, divergence, curl,
  goldenSection, gradientDescent
} from 'math4js/calculus';

// 微分
derivative(x => x ** 3, 1);        // ~3 (f'(x) at x=1)
gradient((x, y) => x ** 2 + y ** 2, [1, 1]); // [2, 2]

// 積分
integral(x => x ** 2, 0, 1);       // 1/3
simpson(x => x ** 2, 0, 1, 100);  // ~0.333

// 多變數
const F = (x, y, z) => [y, -x, 0];
curl(F, 1, 2, 3);                  // [0, 0, -2]

// 優化
goldenSection(x => x ** 2 - 2, 0, 2, 1e-10); // ~1.414
gradientDescent(
  ([x, y]) => (x - 2) ** 2 + (y - 3) ** 2,
  ([x, y]) => [2 * (x - 2), 2 * (y - 3)],
  [0, 0], 0.1, 1e-6
); // ~[2, 3]
```
