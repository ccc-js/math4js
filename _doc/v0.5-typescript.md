# v0.5 - JS to TypeScript

## 目標

將 math4js 所有 JavaScript 程式碼轉換為 TypeScript，並加入完整的型別註記。

## 完成項目

### 轉換的模組

| 模組 | 檔案 |
|------|------|
| statistics | `random.ts`, `stats.ts`, `hypothesis.ts`, `info.ts`, `interval.ts`, `theorem.ts`, `distributions.ts`, `index.ts` |
| linear_algebra | `vector.ts`, `function.ts`, `matrix.ts`, `theorem.ts`, `index.ts` |
| plot | `rplot.ts` |
| 類型宣告 | `src/types.d.ts` |

### 型別定義

```typescript
// 統計結果介面
interface Summary {
  Min: number;
  Q1: number;
  Median: number;
  Mean: number;
  Q3: number;
  Max: number;
  SD: number;
  Var: number;
  N: number;
}

// 矩陣分解結果
interface SVDResult { U: Matrix; S: number[]; V: Matrix; }
interface QRResult { Q: Matrix; R: Matrix; }
interface LUResult { L: Matrix; U: Matrix; P: Matrix; }
interface EigResult { values: number[]; vectors: Matrix; }
```

### 測試轉換

所有 Jest 測試從 `.js` 轉換為 `.ts`：
- 使用 `import` 取代 `require`
- 加入型別註記
- 更新 Jest 配置 (`ts-jest`)

### 設定更新

- `tsconfig.json`: 嚴格模式、宣告輸出
- `.eslintrc.js`: TypeScript ESLint 解析器
- `package.json`: Jest + ts-jest 配置
