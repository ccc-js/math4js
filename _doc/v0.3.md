# v0.3 - 繪圖 (Plot)

## 目標

使用 plotly.js 實現 R 風格的繪圖 API，與 math4py/plot 的設計保持一致。

## 功能

### 核心繪圖函數

| 函數                     | 說明                         |
| ------------------------ | ---------------------------- |
| `plot(x, y, options)`    | 散點圖/折線圖（仿 R plot()） |
| `hist(x, options)`       | 直方圖（仿 R hist()）        |
| `boxplot(data, options)` | 盒鬚圖（仿 R boxplot()）     |
| `qqnorm(x, options)`     | Q-Q 常態圖（仿 R qqnorm()）  |

### 設備控制

| 函數                     | 說明                        |
| ------------------------ | --------------------------- |
| `png(filename, options)` | 打開 PNG 設備（仿 R png()） |
| `pdf(filename, options)` | 打開 PDF 設備（仿 R pdf()） |
| `dev.off()`              | 關閉設備（仿 R dev.off()）  |

### 選項（options）

- `type`: "p"(點), "l"(線), "b"(兩者)
- `main`: 標題
- `xlab`, `ylab`: 軸標籤
- `col`: 顏色
- `xlim`, `ylim`: 軸範圍

## 實作

- 參考 math4py/src/math4py/plot/rplot.py 的設計
- 使用 plotly.js 作為底層渲染引擎
- 提供 R 風格的 API 介面
