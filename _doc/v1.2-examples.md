# v1.2 - 範例 (Examples)

## 目標

為每個模組建立使用範例，展示 API 用法，方便使用者學習與參考。

## 範例結構

```
examples/
├── number_theory.js   # 數論範例
├── linear_algebra.js  # 線性代數範例
├── ndarray.js          # 陣列範例
├── statistics.js       # 統計範例
├── optimization.js     # 優化範例
├── calculus.js        # 微積分範例
├── geometry.js        # 幾何範例
└── algebra.js         # 代數範例
```

## 執行方式

```bash
cd examples
npm run test           # 執行全部範例
npm run number_theory  # 執行單一模組範例
```

## 匯入方式

使用 `package.json` 的 `exports` 欄位設定路徑，可直接用套件名稱匯入：

```javascript
import { gcd, lcm } from 'math4js/number_theory';
import { Matrix } from 'math4js/linear_algebra';
import { Polynomial, horner } from 'math4js/algebra/polynomial';
import { Complex } from 'math4js/algebra/complex';
import { Rational } from 'math4js/algebra/rational';
import { derivative, gradient } from 'math4js/calculus/derivative';
import { trapezoid, simpson } from 'math4js/calculus/integral';
import { hessian } from 'math4js/calculus/multivariable';
import { goldenSection } from 'math4js/calculus/optimize';
import { ndarray, zeros, ones } from 'math4js/ndarray';
import * as ops from 'math4js/ndarray/operations';
import { mean, median, sd, rnorm, t_test_two } from 'math4js/statistics';
import { hillClimbingSimple, simplexMethod } from 'math4js/optimization';
import { Point2D, Vec2, Sphere } from 'math4js/geometry';
```

## 範例內容

### number_theory.js

```javascript
import { gcd, lcm, isPrime, primesUpTo, primeFactors, eulerPhi,
         modPow, modInv, fibonacci, factorial, combinations,
         bezoutIdentity, chineseRemainderTheorem, fermatLittleTheorem } from 'math4js/number_theory';

gcd(48, 18);                              // 6
isPrime(17);                             // true
primesUpTo(20);                           // [2, 3, 5, 7, 11, 13, 17, 19]
primeFactors(18);                         // [2, 3, 3]
eulerPhi(12);                             // 4
modPow(3, 17, 5);                         // 3
modInv(3, 7);                             // 5
fibonacci(10);                            // 55
factorial(6);                             // 720
combinations(10, 3);                     // 120
chineseRemainderTheorem([3, 4, 5], [2, 3, 1]); // 11
```

### linear_algebra.js

```javascript
import { Matrix } from 'math4js/linear_algebra';
import { norm_vector, dot_product, cross_product, matrix_multiply, transpose, trace } from 'math4js/linear_algebra';

const A = new Matrix([[1, 2, 3], [4, 5, 6]]);
const B = new Matrix([[7, 8], [9, 10], [11, 12]]);

matrix_multiply(A.toArray(), B.toArray());
// [[58, 64], [139, 154]]

dot_product([1, 2, 3], [4, 5, 6]);  // 32
cross_product([1, 2, 3], [4, 5, 6]); // [-3, 6, -3]
norm_vector([3, 4]);                  // 5

new Matrix([[1, 2], [2, 5]]).eig().values; // [5.828, 0.172]
```

### ndarray.js

```javascript
import { ndarray, zeros, ones, full, eye } from 'math4js/ndarray';
import * as ops from 'math4js/ndarray/operations';

zeros([3]).toArray();           // [0, 0, 0]
ones([2, 3]).toArray();         // [[1, 1, 1], [1, 1, 1]]
full([5], 99).toArray();        // [99, 99, 99, 99, 99]
eye(3).toArray();              // [[1,0,0], [0,1,0], [0,0,1]]

const x = new ndarray([1, 2, 3]);
const y = new ndarray([4, 5, 6]);
ops.add(x, y).toArray();       // [5, 7, 9]
ops.multiply(x, y).toArray(); // [4, 10, 18]
ops.sum(x);                    // 6
```

### statistics.js

```javascript
import { mean, median, variance, sd, dnorm, pnorm, rnorm, dbinom,
         setSeed, random, t_test_two, conf_interval } from 'math4js/statistics';

mean([1, 2, 3, 4, 5]);              // 3
median([1, 2, 3, 4, 5]);            // 3
variance([1, 2, 3, 4, 5]);          // 2
dnorm(0);                           // 0.3989
pnorm(1.96);                        // 0.975
rnorm(3);                          // [隨機常態樣本]
dbinom(5, 10, 0.5);                 // 0.246

setSeed(42);
random(5);                         // [0.252, ...]

t_test_two([1, 2, 3, 4, 5], [2, 3, 4, 5, 6]);
conf_interval([1, 2, 3, 4, 5]);    // { lower, upper, mean, ... }
```

### optimization.js

```javascript
import { hillClimbingSimple, simulatedAnnealing, simplexMethod, isFeasiblePoint } from 'math4js/optimization';

const f = (x) => (x[0] - 2) ** 2 + (x[1] - 3) ** 2;
hillClimbingSimple(f, [0, 0], 0.5, 100);
// { x: [2, 3], value: 0, ... }

const c = [-1, -1];  // 最大化
const A = [[1, 1], [1, 0], [0, 1]];
const b = [4, 2, 2];
simplexMethod(c, A, b);
// { x, objective, feasible, bounded }

isFeasiblePoint([1, 1], A, b);     // true
```

### calculus.js

```javascript
import { derivative, secondDerivative, gradient } from 'math4js/calculus/derivative';
import { trapezoid, simpson } from 'math4js/calculus/integral';
import { hessian } from 'math4js/calculus/multivariable';
import { goldenSection } from 'math4js/calculus/optimize';

const f = (x) => x ** 3 + 2 * x ** 2 - 5 * x + 1;
derivative(f)(2);                   // 15
secondDerivative(f)(2);             // 14

const f2 = (x) => x[0] ** 2 + x[1] ** 2;
gradient(f2)([1, 2]);              // [2, 4]

trapezoid((x) => x ** 2, 0, 1, 100);  // 0.33335
simpson((x) => x ** 2, 0, 1, 100);    // 0.33333

goldenSection((x) => x ** 2 - 4 * x + 1, 0, 4); // ~2
```

### geometry.js

```javascript
import { Point2D, Vec2, Sphere } from 'math4js/geometry';
import { pointToPoint2D, translate2D, rotate2D, scale2D } from 'math4js/geometry';

new Point2D(1, 2).toString();     // "(1, 2)"
pointToPoint2D(new Point2D(1, 2), new Point2D(4, 6)); // 5

const v = new Vec2(1, 2);
v.add(v).toString();              // "(2, 4)"
v.dot(v);                         // 5

translate2D(new Point2D(1, 0), new Vec2(2, 1)); // (3, 1)
rotate2D(new Point2D(1, 0), Math.PI / 2);      // (~0, 1)
scale2D(new Point2D(1, 0), 2);                // (2, 0)

new Sphere(5).radius;            // 5
```

### algebra.js

```javascript
import { Polynomial, horner } from 'math4js/algebra/polynomial';
import { bisection } from 'math4js/algebra/roots';
import { Complex } from 'math4js/algebra/complex';
import { Rational } from 'math4js/algebra/rational';

const p = new Polynomial([1, -5, 6]);  // x² - 5x + 6
p.eval(0);                             // 1
p.eval(2);                             // 0
p.derivative().toString();            // "12x - 5"

horner([1, -5, 6, -1], 2);           // 7

bisection((x) => x ** 2 - 4, 0, 3); // ~2

const c1 = new Complex(3, 4);
c1.add(new Complex(1, -2)).toString(); // "4 + 2i"
c1.mul(new Complex(1, -2)).toString(); // "11 - 2i"
c1.abs();                             // 5

const r1 = new Rational(2, 3);
const r2 = new Rational(3, 4);
r1.add(r2).toString();               // "17/12"
r1.mul(r2).toString();               // "1/2"
```

## 備註

- 範例使用 ES Module (`import`/`export`)
- 透過 `exports` 欄位設定路徑映射
- 需先執行 `npm run build` 編譯 TypeScript
- 某些 API 名稱與預期不同（如 `Complex.mul` 而非 `multiply`）
- `examples/package.json` 設有 `"type": "module"` 啟用 ESM
