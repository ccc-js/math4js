# v0.1 - 統計模組基礎 (Statistics Foundation)

## 目標

實作可設定種子的亂數產生器，並建立統計分布函數。JS 無內建 seed，這是後續統計功能的基礎。

## 參考來源

`/Users/Shared/ccc/project/math4py/src/math4py/statistics/`

## 目錄結構

```
src/math4js/
  __init__.js
  statistics/
    __init__.js      - 匯出點，別名 R
    distributions.js - 分布函數 d*/p*/q*/r*
    random.js        - 可 seed 的 LCG 亂數核心
    stats.js         - 敘述統計
    theorem.js       - 統計定理驗證

    test_random.js
    test_stats.js
    test_theorem.js
    test_distributions.md
    test_stats.md
```

## 功能項目

### 1. 亂數核心 (random.js)
- [ ] `setSeed(n)` - 設定全域亂數種子
- [ ] `getSeed()` - 取得目前種子
- [ ] `random()` - [0, 1) 均勻分布浮點數
- [ ] `randInt(min, max)` - 整數亂數

### 2. 分布函數 (distributions.js)
- [ ] `dnorm`, `pnorm`, `qnorm`, `rnorm` - 常態分布
- [ ] `dt`, `pt`, `qt`, `rt` - t 分布
- [ ] `dchisq`, `pchisq`, `qchisq`, `rchisq` - 卡方分布
- [ ] `df`, `pf`, `qf`, `rf` - F 分布
- [ ] `dbinom`, `pbinom`, `qbinom`, `rbinom` - 二項分布
- [ ] `dpois`, `ppois`, `qpois`, `rpois` - 卜瓦松分布

### 3. 敘述統計 (stats.js)
- [ ] `mean`, `median`, `var`, `sd`
- [ ] `cov`, `cor`, `quantile`, `iqr`
- [ ] `summary` - 完整統計摘要

### 4. 定理驗證 (theorem.js)
- [ ] `central_limit_theorem`
- [ ] `law_of_large_numbers`
- [ ] `chebyshev_inequality`, `chebyshev_verify`
- [ ] `markov_inequality`, `markov_verify`
- [ ] `bernoulli_verify`, `bayes_theorem`
- [ ] `information_entropy`, `mutual_information`
