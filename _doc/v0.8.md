## 功能項目

### 1. 點 (point.ts) ✅

- [x] `Point1D(x)`, `Point2D(x, y)`, `Point3D(x, y, z)`
- [x] `PointND(coords)` - 任意維度
- [x] `distance(p)` - 兩點距離
- [x] `midpoint(p)` - 中點
- [x] `translate(v)` - 平移
- [x] `scale(factor)` - 縮放

### 2. 向量 (vector.ts) ✅

- [x] `Vec2(x, y)`, `Vec3(x, y, z)`, `Vec(dims)`
- [x] `add`, `sub`, `mul` - 向量運算
- [x] `dot(v)`, `cross(v)` - 內積/外積
- [x] `normalize()` - 單位向量
- [x] `length()` - 向量長度
- [x] `project(v)` - 投影
- [x] `reflect(n)` - 反射

### 3. 直線 (line.ts) ✅

- [x] `Line2D(p1, p2)` - 2D 直線
- [x] `Line3D(p1, p2)` - 3D 直線
- [x] `Ray(p, direction)` - 射線
- [x] `Segment(p1, p2)` - 線段
- [x] `containsPoint(p)` - 點是否在線上

### 4. 平面 (plane.ts) ✅

### 5. 圓 (circle.ts) ✅

## API 設計

```typescript
import {
  Point2D, Vec2, Line2D, Circle,
  distance, translate, rotate2D
} from 'math4js/geometry';

const p1 = new Point2D(0, 0);
const p2 = new Point2D(3, 4);
distance(p1, p2); // 5

const c = new Circle(new Point2D(0, 0), 1);
c.area(); // π

const v = new Vec2(1, 0);
rotate2D(v, Math.PI / 2); // Vec2(0, 1)
```
