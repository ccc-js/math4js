## 功能項目

### 1. 有理數 (rational.ts) ✅

- [x] `Rational(num, den)` - 分數類別，自動約分
- [x] `add`, `sub`, `mul`, `div` - 分數四則運算
- [x] `simplify()` - 化簡分數
- [x] `toNumber()` - 轉浮點數
- [x] `equals(r)` - 比較相等

### 2. 複數 (complex.ts) ✅

- [x] `Complex(re, im)` - 複數類別
- [x] `add`, `sub`, `mul`, `div` - 四則運算
- [x] `conj()` - 共軛複數
- [x] `abs()`, `arg()` - 極座標形式
- [x] `exp()`, `log()`, `pow()` - 複數指數對數
- [x] `sqrt()`, `sin()`, `cos()` - 三角函數

### 3. 多項式 (polynomial.ts) ✅

- [x] `Polynomial(coeffs)` - 係數陣列 [a0, a1, a2, ...]
- [x] `add`, `sub`, `mul` - 多項式四則運算
- [x] `degree()`, `coeffs()` - 取得資訊
- [x] `eval(x)` - 求值
- [x] `derivative()` - 導函數
- [x] `integral()` - 積分函數
- [x] `compose(p)` - 合成 f(g(x))

### 4. 求根 (roots.ts) ✅

- [x] `bisection(f, a, b, tol)` - 二分法
- [x] `newton(f, df, x0, tol)` - 牛頓法
- [x] `secant(f, x0, x1, tol)` - 割線法
- [x] `horner(coeffs, x)` - Horner 快速求值

### 5. 分數系統 (fraction.ts)

- [ ] `Fraction` - 分數類別（與 Rational 相同）
- [ ] `parseFraction(str)` - 字串轉分數
- [ ] `continuedFraction(n)` - 連分數展開

## API 設計

```typescript
import { Rational, Complex, Polynomial, bisection, newton } from 'math4js/algebra';

// 有理數
const r1 = new Rational(1, 3);
const r2 = new Rational(2, 6); // 自動約分為 1/3
const r3 = r1.add(r2); // 2/3

// 複數
const c = new Complex(3, 4);
c.abs();    // 5
c.arg();    // atan2(4, 3)

// 多項式
const p = new Polynomial([1, -5, 6]); // x^2 - 5x + 6
p.eval(2);   // 0
p.derivative(); // Polynomial([1, -5])

// 求根
bisection(x => x ** 2 - 2, 0, 2, 1e-10); // ~1.414
newton(x => x ** 2 - 2, x => 2 * x, 1.5); // ~1.414
```
