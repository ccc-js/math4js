## 數學背景

### 無約束優化

梯度下降法是最基本的優化方法，沿負梯度方向迭代：

$$x_{k+1} = x_k - \alpha \nabla f(x_k)$$

牛頓法使用二階資訊加速收斂：

$$x_{k+1} = x_k - H^{-1}(x_k) \nabla f(x_k)$$

共軛梯度法適用於大型稀疏系統，避免儲存完整 Hessian 矩陣。

Adam 結合動量與 RMSProp 的優勢，自適應學習率。

### 爬山法

爬山法是局部搜索策略，簡單但容易陷入局部最優。模擬退火加入機制跳出局部最優：

$$P(\text{accept}) = e^{-\Delta E / T}$$

冷卻 schedule 影響最終解的品質。

### 線性規劃

單形法在可行域頂點之間移動，逐步改善目標函數值。內點法在可行域內部移動，對大規模問題更有效。

## 使用範例

```typescript
import { gradientDescent, hillClimbingSimple, simplexMethod } from './optimization/index.js';

// 梯度下降
const f = (x: number[]) => (x[0] - 2) ** 2 + (x[1] - 3) ** 2;
const result = gradientDescent(f, [0, 0]);
console.log(result.x, result.value); // [2, 3], ~0

// 爬山法
const result2 = hillClimbingSimple(f, [0, 0], 0.5, 100);

// 線性規劃
const c = [-1, -1]; // 最大化
const A = [[1, 1], [1, 0], [0, 1]];
const b = [4, 2, 2];
const lp = simplexMethod(c, A, b);
```
