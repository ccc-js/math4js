# v0.2 - 統計推論 (Statistical Inference)

## 目標

實作假設檢定、區間估計等統計推論功能。

## 參考來源

`/Users/Shared/ccc/project/math4py/src/math4py/statistics/function.py`

## 功能項目

### 1. 假設檢定 (src/statistics/hypothesis.js)

- [ ] `z_test(x, mu0, sigma, alpha)` - Z 檢定
- [ ] `t_test(x, mu0, alpha)` - 單樣本 t 檢定
- [ ] `t_test_two(x, y, alpha)` - 雙樣本 t 檢定
- [ ] `chisq_test(observed, expected)` - 卡方檢定
- [ ] `anova(groups)` - 單因子變異數分析

### 2. 區間估計 (src/statistics/interval.js)

- [ ] `conf_interval(x, alpha)` - 平均值信賴區間
- [ ] `bootstrap_ci(x, statFn, n, alpha)` - Bootstrap 信賴區間

### 3. 資訊理論 (src/statistics/info.js)

- [ ] `entropy(p, base)` - 熵
- [ ] `cross_entropy(p, q, base)` - 交叉熵
- [ ] `kl_divergence(p, q, base)` - KL 散度
- [ ] `mutual_information(x, y)` - 互資訊

### 4. 進階統計函數 (src/statistics/advanced.js)

- [ ] `z_score(x, mu, sigma)` - Z 分數
- [ ] `standardize(x)` - 標準化
- [ ] `log_likelihood(y, y_pred)` - 對數概似
- [ ] `aic(n, log_lik, k)` - AIC
- [ ] `bic(n, log_lik, k)` - BIC
